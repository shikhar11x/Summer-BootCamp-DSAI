## 5. Avengers Face Recognition System

A Deep Learning computer vision project that identifies five Avengers characters from cropped face images using **Transfer Learning with MobileNetV2**.

**Key Highlights**
- Transfer Learning using MobileNetV2
- Image preprocessing and data augmentation
- Face classification with confidence score
- Achieved **79.07% Test Accuracy**
